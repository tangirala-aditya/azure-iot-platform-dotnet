    Set Theme

    API that sets the theme of the solution.

    Permissions

    - ReadAll

    Http Request

    PUT /config/v1/solution-settings/theme

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Schema

    {
  "name": "string",
  "description": "string",
  "azureMapsKey": "string"
}

    Sample Request Body

    {
    "name": "My Solution",
    "description": "My Solution Description",
    "azureMapsKey": "qgZ7OQi8XryD6py8L-ZId8DNuaWtCbdrQ996mY3aQl8"
}

    Response Schema

    {
  "name": "string",
  "description": "string",
  "azureMapsKey": "string"
}

    Sample Response

    {
    "name": "My Solution",
    "description": "My Solution Description",
    "azureMapsKey": "qgZ7OQi8XryD6py8L-ZId8DNuaWtCbdrQ996mY3aQl8"
}

    Security

    Type: OpenID Connect
    Flow: implicit
    Authorization URL: https://login.microsoftonline.com/crsltestab2c.onmicrosoft.com
