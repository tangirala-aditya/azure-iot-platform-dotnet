    List Alarms

    Service: Device Telemetry

     GET https://{platform-url}/telemetry​/v1​/Alarms

    URI Parameters

    Name Required Type Description
    from string Start time as ISO8601 or "NOW-{ISO 8601 duration}"
    to string End time as ISO8601 or "NOW-{ISO 8601 duration}"
    order string Set sort order: oldest ("asc"), newest ("desc")
    skip string Alarms to skip. Used for pagination.
    limit string Return count. Default 100.
    devices string Filter used to request specific devices

    Responses

    Name Type
    200 OK [NameOfResponse]
