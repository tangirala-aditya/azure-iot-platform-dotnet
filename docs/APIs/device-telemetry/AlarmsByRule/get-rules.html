    Get rules with alarms count

    API that fetches the rules details with a count of recorded alarms.

    Permissions

    ReadAll

    Http Request

    GET /telemetry/v1/AlarmsByRule

    Query parameters

    Query parameter Type Example Description
    from string NOW-PT24H The start of the time window for the query. ISO8601 format or "NOW-{ISO 8601 duration}"
    to string NOW The end of the time window for the query. ISO8601 format or "NOW-{ISO 8601 duration}"
    order string desc Whether to sort the result from the oldest (asc) or the most recent (desc)
    skip int 0 How many records to skip, used to paginate through the global list of rules
    limit int 50 How many records to return, used to paginate through the global list of rules
    devices string id1,id6,xyz A filter used to request rules for specific devices

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body

    N/A
