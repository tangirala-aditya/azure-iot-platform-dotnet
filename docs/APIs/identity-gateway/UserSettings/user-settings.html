    User Settings of input User

    This Method returns the settings of input userId

    Permissions

    **ReadAll** permission is required to call this api.

    Http Request

    GET /auth/v1/settings/{userId}/all

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body

    N/A

    /auth/v1/settings/3e30f301-4f57-4b3e-949a-3ff8e93870b5/all
