    Get Deployment

    Gets the details of specifed deployment from the storage or from IoTHub based on the state.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    GET /iothub-manager/v1/deployments/{id}?includeDeviceStatus={includeDeviceStatus}&isLatest={isLatest}

    Request Headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Query Params

    Name Value Default Value
    includeDeviceStatus Value of true will add data to the Device Status reflecting the status of Deployment. false
    isLatest true will retrieve the data of Deployment from IoTHub reflecting current status.
    false will retrieve the data from storage if deployment is not currently impacting the targetted device group.    		 true

    Request Body

    NA
