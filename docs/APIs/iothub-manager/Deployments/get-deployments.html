    Get Deployments

    Lists the deployments from the storage.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    GET /iothub-manager/v1/deployments

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Request Body

    NA

    Response

    If successful, this method returns DeploymentListApiModel object in the response body.
