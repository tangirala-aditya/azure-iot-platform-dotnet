    Get Device Deployment History

    Retrieves the DeviceTwins properties from the Deployment Impacted Devices collection for the device.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    GET /iothub-manager/v1/devices/deploymentHistory/{id}

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Query Params

    Name Value
    id Device Id

    Request Body

    NA

    Response

    If successful, this method returns TwinPropertiesListApiModel object in the response body.
