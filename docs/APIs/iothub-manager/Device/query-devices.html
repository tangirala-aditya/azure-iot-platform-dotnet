    Query Devices

    Lists the Devices based on the query conditions from request body.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    POST /iothub-manager/v1/devices/query

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json
    x-ms-continuation Continuation Token from response of previous request,if request is called to get next dataset.

    Request Body

    JSON serialized string representing the conditions to fetch the devices.

    Response

    If successful, this method returns DeviceListApiModel object in the response body.
