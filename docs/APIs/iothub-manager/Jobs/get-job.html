    Get Job

    Retrieves the details of an jon from IoTHub based on jobid, job status.

    Permissions

    ReadAll permission is required to call this api.

    Http Request

    GET /iothubmanager/v1/jobs/{id}?includeDeviceDetails={includeDeviceDetails}&deviceJobStatus={deviceJobStatus}

    Request headers

    Header Value
    Authorization Bearer {token}. Required.
    Content-Type application/json

    Query Params

    Name Value
    id Job Id
    includeDeviceDetails true, will include Devices as part of result.
    deviceJobStatus Pending = 0
    Scheduled = 1
    Running = 2
    Completed = 3
    Failed = 4
    Canceled = 5

    Request Body

    NA

    Response

    If successful, this method returns JobApiModel object in the response body.
